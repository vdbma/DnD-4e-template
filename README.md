# DnD 4 template

[![pre-commit](https://img.shields.io/badge/pre--commit-enabled-brightgreen?logo=pre-commit&logoColor=white)](https://pre-commit.com/) [![made-with-latex](https://img.shields.io/badge/Made%20with-LaTeX-1f425f.svg)](https://www.latex-project.org/)


This LaTeX template is inspired from the PHB of the 4th edition of one of the most famous roleplaying games.

## Usage

This template requires XeLaTex.

## Contributions welcome

Feel free to contribute (via pull requests) too correct or improve this template !
